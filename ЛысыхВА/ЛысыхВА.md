## Краткое описание покрытого тестами функционала

Был добавлен тест для функции активации локализации (`activate()`).

Проверялось:

- корректное подключение локали при явном указании пути к каталогу переводов;
- создание объекта перевода;
- установка текущей локали;
- корректная деактивация локали через `deactivate()`.

Добавленный тест позволил покрыть ранее непокрытую ветку кода, связанную с активацией локализации.
